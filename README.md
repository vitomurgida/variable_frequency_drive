# variable_frequency_drive
A simulator of a 6 pulses VFD

# Introduction and motivation
A variable frequency drive is a device allowing to control the frequency and power supplied to a rotating electromagnetic machine such as a permanent magnet electric motor.

This code allows to simulate current and voltage across the different parts of a variable frequency drive (rectifier diodes, capacitor DC bus, PWM transistors, electric motor). The FFT of the different signals is computed and the frequency spectra are generated, allowing for a quantification of the signal distortion. 

As a further step, magnetic forces are assumed to depend quadratically from the currents, while acoustic noise generated by components is assumed to depend directly to magenetic forces. In such a way, acoustic harmonics can be also estimated, helping troubleshooting related to noise generated by electric components.

# Input
- electric engine rpm
- electric engine number of poles
- electric engine average power
- line-to-line rms input voltage
- system electrical frequency
- system resistance
- filter inductance
- DC bus capacitance
- diode threshold voltage
- motor resistance
- motor inductance
- PWM microprocessor sampling frequency
- simulated time
- FFT parameters

# Output
The following plots are output:
- current out from diodes
- voltage on DC bus
- IGBT transistors switching history
- three phase currents on electrical motor
- acoustic harmonics (squared current)
